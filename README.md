# COVID-19 Data Exploration Using SQL Server  
A data analysis project using SQL Server to explore global COVID-19 trends, including infection rates, death percentages, and vaccination progress. This project demonstrates practical SQL skills used in real-world data analysis, database management, and BI workflows.

---

## Features
- Data cleaning & filtering  
- Infection, death, and population impact analysis  
- Vaccination progress with rolling metrics  
- Global trends summarized using aggregates  
- SQL views, temp tables, CTEs, and window functions  
- Ready for integration with Tableau or BI dashboards
